Európai utazásaink - Projektfeladat

Üdvözlünk az utazási projektünkben! Ezt az oldalt azért hoztuk létre, hogy két kedvenc úticélunkat, a horvátországi Rovinjt és a cseh fővárost, Prágát mutassuk be. A célunk az volt, hogy egy esztétikus, könnyen kezelhető felületet építsünk, ahol a látogatók kedvet kaphatnak egy kis felfedezéshez.

Funkciók:
  - Interaktív, letisztult navigációs menü
  - Reszponzív design
  - Modern tipográfia: Google Font integráció a jobb olvashóságért

Felhasznált technológiák:
  - HTML & CSS
  - Google Fonts
  - Grid (Az oldal elrendezéséhez)
  - Media Queries (Számítógépre és mobilra is optimalizált)
  - Git & GitHub használata a közös munkához
